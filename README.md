### 🐾 **개요**

**“사랑하는 반려동물에게 최상의 의료서비스를 제공합니다.”**  
간편한 진료 예약부터 체계적인 사후관리까지,  
**신뢰할 수 있는 동물병원 웹사이트**를 제공합니다.

-   반려동물을 위한 맞춤형 진료 예약 시스템
    
-   진료 이력 확인 및 사후관리 기능 제공
    
-   사용자 친화적인 인터페이스와 간편한 접근성

💡 *"HI ! Pet"은 반려동물을 위한 따뜻한 경험을 제공하는 병원 웹사이트입니다.*

----------


### 🛠️ **설계의 주안점**

1.  **직관적인 UI/UX**  
     - 누구나 쉽게 이용할 수 있는 **사용자 친화적 인터페이스** 구현
    
2.  **권한에 따른 페이지 분리**  
     - **회원 / 비회원 / 관리자** 각각에 맞는 **전용 페이지** 설계
    
3.  **세분화된 예약 시스템**  
     - 반려동물 증상에 따라 **전문 수의사를 선택**할 수 있는 보호자 맞춤형 예약 구조
    
4.  **커뮤니티 기능 강화**  
     - 반려동물의 건강정보를 공유하고 **의견을 나눌 수 있는 게시판 서비스** 제공
    
5.  **지속적인 의료 상담**  
     - 진료 이후에도 **1:1 질문 게시판**을 통해 전문가와 지속적인 **의료 상담 가능**


----------

### 📌 **프론트엔드 기능 정리**


| 구분       | 내용 |
|------------|------|
| **홈페이지** | 병원 메인 페이지 접근 포털 |
| **메인 페이지** | 병원 로고, 로그인/회원가입 가능, 로그인 시 메인으로 리다이렉트 |
| **메뉴 구성** | 병원 소개, 진료 안내, 고객센터, 건강 정보 |
| **병원 소개** | 병원 개요, 위치, 찾아오는 방법 (지도 API 활용) |
| **진료 안내** | DB에서 진료 리스트 조회, hover 효과, 예약 페이지 이동, 예약 중복 방지 |
| **고객센터** | 공지사항, 1:1 문의 (비회원 접근 시 로그인 유도, 문의 비공개 처리) |
| **건강 정보** | 건강 정보 영상, 파일 게시, 관리자 CRUD 가능 |
| **관리자 페이지** | 전체 회원/예약/문의 관리, 공지사항/건강정보 CRUD, 비회원 예약 관리, 휴무일 설정 등 |
| **회원 기능** | 정보 수정, 본인 예약 현황 확인 및 취소 가능 |

----------

### ⚙️ **백엔드 기능 정리**


| 구분       | 기능 내용 |
|------------|-----------|
| **Spring Security** | 접근 경로 설정, 비밀번호 암호화 (BCrypt), 인증 매니저 설정 |
| **JWT 토큰** | 로그인 시 JWT access/refresh 토큰 발급, 유효성 검증 및 예외 처리 |
| **회원가입** | 아이디 중복 확인, 가입 처리 |
| **로그인** | 아이디/비밀번호 확인, 토큰 발급 |
| **공지사항** | 전체 조회, 키워드 검색, 관리자 전용 CRUD |
| **1:1 문의** | 토큰 인증, 본인 문의 출력, CRUD 가능, 비공개 처리 |
| **진료 안내/예약** | 회원/비회원 구분 저장, 예약 가능 시간 조회, 예약 처리 및 중복 방지 |
| **건강 정보** | 관리자 CRUD (게시글, 파일, 영상), 회원 댓글 기능 |
| **회원 기능** | 정보 수정, 예약 내역 조회 |
| **관리자 기능** | 모든 회원/비회원 정보 조회 및 검색, 1:1문의/댓글/예약 관리, 휴무일 설정, 건강 정보 관리 |
